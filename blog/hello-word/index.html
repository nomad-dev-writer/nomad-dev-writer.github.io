Hello world

Posted on 2024-08-18

This is definitely not the first time I tried to write and run a blog. Every single time, they failed for... well countless number of reasons. One of the main reasons I failed was because obviously I was over-thinking, trying to be perfect... in everything. Starting from the choice of blogging platform and technology, the design, the theme, and also the post itself. Endless considerations. My design and tech background definitely did not help there.

Oh well, here I go again, after nth failed attempt to run a blog, ready to face (n+1)th failure. At least this time I have a little more advantage, as there are so many LLMs out there to help me. Plus, I've set some rules to myself this time on how I want to run this blog:

  • Write fast, fix later. As hinted in the about page, all the posts will be frequently updated.
  • Broken stuff? I don't care. I'll stop falling into the rabbit hole of infinitely fixing website. I'm supposed to be posting, not coding.
  • Do I sound stupid? Just be myself. Nobody is perfect and we all have different opinions and views towards this world. I'll try best to be honest, although I'll still be cautious and think twice on the topic.

Let's see how long I'll manage to run this site this time. My first aim is to post five notes, without thinking much.
